Time series anomaly detetection

This model can detect anomalies, i.e. distinguish certain data points that do not follow the same trend as the rest of the data. It identifies in red the data that is considered to be different, an anomaly.

Model: Timeseries anomaly detection using an Autoencoder
Data: Numenta Anomaly Benchmark (NAB)
Example from: Dheiver Santos on Huggingface
