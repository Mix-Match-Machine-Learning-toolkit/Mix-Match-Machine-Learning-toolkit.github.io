Song recommender

This example will recommend new songs based on your playlist, a song or artist top tracks. To create the recommendations it uses certain features of the song, for example artist, release date and tempo. It therefore does not directly bases it recommendations on the audio itself but features derived from it.

Model: Recommender system
Data: Spotify Million Playlist Dataset
Example from: Abdelrhman elruby on Huggingface
