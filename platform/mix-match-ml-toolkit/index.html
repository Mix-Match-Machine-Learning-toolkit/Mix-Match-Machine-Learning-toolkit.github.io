Mix & Match ML toolkit

Description:

The Mix & Match ML toolkit can be used to learn more about ML and/ or during the ideation process. The tokens will give you an explanation and inspiration about how ML works and can be used in design.

Design phase:

Ideation

Audience:

Designers, non ML experts

Required knowledge:

Optional: design background

User guide

With existing toolkit
With website only
DIY toolkit
Custom examples/ website
With existing toolkit
  1. Turn on the toolkit
  2. Go to a-jansen.github.io
  3. Connect the toolkit with the website via Bluetooth
  4. Place the tokens on the sensor board to learn and get inspired
With website only
  1. Go to a-jansen.github.io
  2. Click “skip connection”
  3. Use the menu to navigate
DIY toolkit
  1. Go to https://github.com/A-Jansen/Mix-Match-ML-toolkit
  2. Laser cut the tokens and box
  3. Assemble the tokens and box
  4. Solder the electronics
  5. Initialize the token ID’s
  6. Place the electronics inside the box
Custom examples/ website
  1. Go to https://github.com/A-Jansen/Mix-Match-ML-toolkit
  2. Clone the code for the website
  3. Change the data in the XML files
  4. Run on a local host or with your own hosting
